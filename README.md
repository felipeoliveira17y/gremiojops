# 🗳️ Grêmio JOPS - Sistema de Votação

Este é um sistema de votação digital moderno e seguro, desenvolvido para as eleições do Grêmio Estudantil JOPS. A aplicação foi construída focando em uma experiência de usuário fluida (UI/UX) e integridade dos dados.

## 🚀 Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema Web:

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Banco de Dados & Autenticação:** [Supabase](https://supabase.com/)
- **Ícones:** [Lucide React](https://lucide.dev/)

## ✨ Funcionalidades

### Para Eleitores:
- **Acesso Seguro:** Login via matrícula e data de nascimento.
- **Interface Intuitiva:** Seleção de chapas com cards modernos e efeitos visuais.
- **Validação:** Sistema que impede votos duplicados (um voto por matrícula).
- **Voto Nulo:** Opção de voto nulo integrada de forma clara.

### Para Administradores (Docentes):
- **Painel de Controle:** Visualização de estatísticas em tempo real.
- **Relatórios:** Acompanhamento da lista de eleitores e participação.
- **Segurança:** Acesso restrito apenas para usuários com permissão de administrador.
